# NaturalStyleTransfer-LBFGS
Repositori ini berisi implementasi Natural Style Transfer (NST) berbasis arsitektur VGG19 dan metode optimasi L-BFGS. Proyek ini bertujuan menggabungkan gambar content dan gambar style untuk menghasilkan visual akhir yang artistik. Dibuat sebagai bagian dari tugas individu mata kuliah Deep Learning Lanjut.
